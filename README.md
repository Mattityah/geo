# Gioco-Geometry

**A browser-based image effects lab by [GiocoStudio](https://github.com/Mattityah)**

---

## About

Gioco-Geometry is a creative tool for transforming images into geometric and algorithmic compositions — halftone grids,screen simulations, ASCII art, and more. Everything runs directly in the browser, no installation required.

The app is built around the idea that technology should serve creativity, not the other way around. Upload an image, explore the parameters, export what you find.

---

## Origin

This project is part of **GiocoStudio** — a space dedicated to exploration and the making of things. *Gioco* is that spirit is at the center of everything here: the belief that through play and curiosity, technology becomes a tool for expression rather than a barrier to it.

## Effects

| Effect | Description |
|--------|-------------|
| **Halftone** | Geometric grid with Floyd-Steinberg / Bayer dithering, multiple shapes, multi-color palette, Benday grid, relief, texture modulation |
| **CRT** | Phosphor dot simulation — Monitor, TV, and LCD modes with bloom, glow, and RGB convergence |
| **ASCII** | Character-density rendering with custom charset, borders, and clipboard export |

---

## Features

- Upload any image — PNG, JPG, WebP
- Real-time parameter controls with live preview
- Image transform: scale, position, rotation
- Size presets: Instagram, 16:9, Story, A4, Banner — or custom dimensions
- Export as **PNG**, **JPG** (with quality control), or **SVG** (Halftone)
- Save and load **presets** as `.json` files
- Style presets: Micro-Geo, Medieval, Pastel, Bold
- Size texture modulation: Wave, Noise, Voronoi
- Mobile-first layout
